# Azure Documentation

This repository contains Azure cloud platform learning documentation covering services, tools, and best practices.

## Files

- **Basics.md** - Azure fundamentals, subscriptions, resource groups, VMs, storage, networking
- **Azure-Compute.md** - Compute services including VMs, scale sets, App Service, Functions
- **Azure-Storage.md** - Storage accounts, Blob, File Shares, Queues, Table storage
- **Azure-Networking.md** - VNets, NSGs, load balancing, VPN, ExpressRoute, DNS
